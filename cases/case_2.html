实施亮点

1、API+H5 混合架构
国内机酒 API 接入，车(已不合作)火 H5 测试，防重机制、映射逻辑、价格同步全跑通。

2、Mapping 准确率与资源露出拉满
酒店数据 Mapping 优化，确保协议价稳定、房型完整、低价优先展示。

3、国际机票业务前置备战
专属国际客服群、紧急预订流程、补单 SOP，虚拟行程单以真实出票替代。手工补单

4、全场景回归测试
联合用友 + 客户 + 集成三方，提前解决下单失败、重复订单、价格波动等卡点。

5、常态化资源巡检
核心航司（东航 / 山航）、核心酒店（汉庭 / 全季）每日巡检，保障协议价在线。

6、线下客服组驻场支持
内部宣导 + 使用培训，确保员工快速上手、问题即时解决

7、同屏比价 低价优先
